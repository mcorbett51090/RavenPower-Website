# Raven Power Website

**Raven Power Website Design System** — Specialized agents for building high-quality websites for small businesses and professional services.

Supports both no-code (Webflow) and modern code-based (Next.js + Tailwind) paths with clear decision gates.

## Structure

- `docs/` — Planning documents, proposals, and decision frameworks
- `plugins/website-design/` — RavenClaude-compatible plugin with specialist agents for web projects

## Key Components

- Client intake and proposal templates
- Decision gates for choosing build paths
- Dedicated agents: Web Strategist, UI/UX Designer, Information Architect, Frontend Engineer, Integration Specialist, SEO & Performance, QA & Launch

## Usage

This repository is designed to be used with **RavenClaude**.
Install the `website-design` plugin to get access to the full set of specialist agents for website projects.

## RavenClaude Integration

Part of the growing RavenClaude ecosystem for domain-specific agent teams.
