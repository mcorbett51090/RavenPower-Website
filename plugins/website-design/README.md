# website-design Plugin

Specialist agents for building high-quality websites.

## Key Feature: Decision Gate

This plugin enforces a critical early decision:

- **Path A**: Client wants easy self-editing → Webflow (recommended for most small businesses)
- **Path B**: Client is okay with code → Next.js + Tailwind + Vercel

The **Web Strategist** is responsible for facilitating this decision before any other work begins.

## Agents

- `web-strategist.md` — Entry point + Decision Gate
- `information-architect.md` — Sitemaps, navigation, user flows

More agents coming soon (UI/UX Designer, Frontend Engineer, etc.).
